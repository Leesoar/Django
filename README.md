# 准备

* **Step 1**：创建虚拟环境
```OS
	python -m virtualenv env_name
```

* **Step 2**：触发/激活虚拟环境
```OS
	cd env_name\Scripts\activate.bat
```

* **Step 3**：创建项目
```OS
	cd env_name
	django-admin startproject project_name
```

* **Step 4**：运行
```OS
	cd project_name
	python manage.py runserver [port](可选-设置端口号)
```
